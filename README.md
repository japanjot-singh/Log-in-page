# 🔐 Java Login & Registration System (JDBC + AWT)

## 📌 Overview

This project is a **desktop-based authentication system** built using **Java AWT and JDBC**.  
It allows users to **create an account and log in**, with data stored in an **Oracle database**.

The application demonstrates the integration of **GUI + Database connectivity + user authentication logic**.

---

## 🚀 Features

- ✅ User Registration (Create Account)  
- ✅ User Login Authentication  
- ✅ Oracle Database Connectivity using JDBC  
- ✅ GUI built using Java AWT  
- ✅ Input validation (username already taken, invalid login)  
- ✅ Password masking using echo character  

---

## 🛠️ Tech Stack

- **Language:** Java  
- **GUI:** AWT  
- **Database:** Oracle (XE)  
- **Connectivity:** JDBC  
- **SQL Operations:** SELECT, INSERT  

---

## 🧩 Project Modules

- **Main Frame**
  - Choose between Login or Create Account  

- **Create Account (FrameCR)**
  - Enter username & password  
  - Checks if username already exists  
  - Stores new user in database  

- **Login (FrameLO)**
  - Validates user credentials  
  - Displays success or error message  

- **Welcome Screen (FrameW)**
  - Displayed after successful login/signup
